# what i'm on

![Project Banner](assets/banner.png)

A personal space to showcase your current obsessions and crowdsource your next favorite thing. Built for Shift APPens 2026. Now with cross-media search across movies, TV shows, books, music, podcasts, albums, and games.

## Overview

Share what you're into and get curated recommendations. A community-driven platform where users post their current interests and passions, while discovering new favorites from others. Built with Next.js and Supabase for secure authentication and seamless user experience.

## Tech Stack

- Next.js 15+ (App Router)
- Supabase (Authentication and Backend)
- TypeScript
- Tailwind CSS
- Search APIs: TMDB, Google Books, iTunes, RAWG

## Getting Started

### Prerequisites

- Node.js 18+ installed
- Supabase account and project configured
- Copy `.env.example` to `.env.local` and fill in the required variables

### Installation

Install dependencies:

```bash
npm install
```

### Development Server

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
